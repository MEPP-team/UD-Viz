Frame3DBase

new Frame3DBase(options, init3Dopt)

Basic Frame3D wrap different html element to handle CSS3D rendering CSS3DRenderer. It's possible to add Billboard to this. Composed with THREE.Scene + THREE.PerspectiveCamera + THREE.WebGLRenderer.

Parameters:
NameTypeAttributesDefaultDescription

options to configure frame3Dbase

Properties

catchEventsCSS3Dboolean<optional>
false

true

THREE.Scene + THREE.PerspectiveCamera + THREE.WebGLRenderer should be init
